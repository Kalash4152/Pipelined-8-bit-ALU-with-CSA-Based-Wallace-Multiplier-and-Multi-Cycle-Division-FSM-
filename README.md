# Pipelined-8-bit-ALU-with-CSA-Based-Wallace-Multiplier-and-Multi-Cycle-Division-FSM-
Design and implementation of a 4-stage pipelined 8-bit ALU in Verilog featuring fast arithmetic/logic operations, a carry-save (Wallace-style) pipelined multiplier, and a multi-cycle restoring division FSM. Includes flag handling, valid signaling, and latency-aware datapath design.
